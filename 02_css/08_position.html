Position

static
기본값. 문서 내에 작성된 순서대로 화면 배치. offset(top | right | bottom | left) 값 무시.
relative
static처럼 문서 내에 작성된 순서대로 화면 배치되지만, offset을 적용 가능.
absolute
기준이 되는 부모요소(position:static이 아닌 가장 가까운 부모)로부터 offset값을 적용.
fixed
viewport 기준으로 offset값 적용
